# Code Review с GitLens

## Процесс code review

### 1. Просмотр изменений
- `GitLens: Compare File with...` - сравнить файл с другой веткой
- `GitLens: Show Commit Details` - детали коммита

### 2. Изучение истории
- Наведите курсор на строку → увидите автора
- Кликните на GitLens аннотацию → история изменений
- `GitLens: Show Line History` - история конкретной строки

### 3. Сравнение веток
```bash
# В GitLens панели:
# Branches → Выбрать ветку → Compare with main
```

### 4. Полезные команды
- `Ctrl+Shift+P` → `GitLens: Open Changes`
- `Ctrl+Shift+P` → `GitLens: Show Quick Commit File Details`

## Горячие клавиши

| Команда | Действие |
|---------|----------|
| `Alt+B` | Toggle Git Blame |
| `Alt+H` | Show File History |
| `Alt+C` | Show Commit Graph |
